Zur Installation müssen die beiden Dateien (UH Spielanzeige Setup und UH Spielanzeige) heruntergeladen werden

Nun kann UH Spielanzeige Setup ausgeführt werden. Wenn .Net Runtime 6.0.0 noch nicht auf dem System installiert ist, wird dies nun gemacht.
Wenn es schon installiert war, wird der Installer geöffnet, der die Applikation installiert und eine Verknüpfung auf dem Desktop einrichtet.
